# SAP O2C Analytics Dashboard
Order-to-Cash Dashboard Using Simulated Data Products

## 📌 Overview
This project demonstrates SAP Data Analytics concepts applied to the Order-to-Cash (O2C) cycle using a synthetic dataset. Since direct SAP system access was unavailable, simulated transactional data was created to represent SAP-style data products.

The project uses Python and Streamlit to provide an interactive dashboard with KPIs, revenue analysis, sales trends, and customer insights.

## 🚀 Features
- Total Revenue KPI
- Average Order Value KPI
- Paid Orders Percentage
- Revenue by Region
- Monthly Sales Trend
- Top 5 Customers by Revenue
- Paid vs Unpaid Orders
- Interactive filters by Region and Customer

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib
- Streamlit

## ⚙️ Setup Instructions

### Install dependencies
```bash
pip install -r requirements.txt
```

### Run Streamlit dashboard
```bash
python -m streamlit run dashboard_app.py
```
